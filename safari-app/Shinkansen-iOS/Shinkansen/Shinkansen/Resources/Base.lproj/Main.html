Shinkansen

快速、流暢的網頁與字幕翻譯

Shinkansen 是 Safari 的網頁翻譯擴充功能。這個 App 是 iOS host，擴充功能本身要在 Safari 設定中啟用後才會運作

啟用方式

  1. 在 iOS「設定」中找到「Safari」→「擴充功能」，開啟「Shinkansen」並允許網站存取權限
  2. 或在 Safari 網址列點擴充功能按鈕，選「管理擴充功能」開啟

Fast web & subtitle translator

Shinkansen is a Safari web translation extension. This app is the iOS host — the extension itself must be enabled in Safari Settings before it works.

How to enable

  1. In iOS Settings, open Safari → Extensions, turn on “Shinkansen” and allow website access
  2. Or in Safari, tap the extensions button in the address bar and choose “Manage Extensions”
