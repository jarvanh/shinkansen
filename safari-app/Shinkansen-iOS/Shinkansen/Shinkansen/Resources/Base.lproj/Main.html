Shinkansen

快速、流暢的網頁與字幕翻譯

Shinkansen 是 Safari 的網頁翻譯擴充功能

以下幾個步驟完成設定

填入 Gemini API Key

使用 Gemini 翻譯能提供最自然流暢的語感。API Key 只存在這台裝置，不會上傳

目前還沒有 API Key 也沒關係，現在可以先不用輸入，以後再申請，下一步選擇免費的 Google 翻譯即可

取得免費 API Key

選擇預設翻譯方式

點懸浮按鈕翻譯

開啟網頁後，畫面右下角會出現懸浮翻譯按鈕，輕點一下就翻譯整頁，再點一次切回原文。按鈕可以拖曳移動、在設定中調整大小或關閉

四指輕點（替代方式）

四指輕點預設關閉。如果不喜歡畫面上的懸浮按鈕，可以在設定中關閉懸浮按鈕、改開四指手勢，之後用四根手指輕點任何網頁就能翻譯整頁，再點一次切回原文

YouTube 字幕自動翻譯

打開 YouTube 影片時，Shinkansen 會自動把字幕翻成中文。不需要時可在擴充功能設定中關閉

最後一步：在 Safari 啟用

啟用方式

  1. 點下方按鈕開啟「設定」，前往「Safari」→「擴充功能」
  2. 開啟「Shinkansen」，並把網站存取權限設為「允許」（建議選「所有網站」）

也可以在 Safari 網址列點擴充功能按鈕，選「管理擴充功能」開啟。剛才的設定會在啟用後自動同步到擴充功能

Fast web & subtitle translator

Shinkansen is a Safari web translation extension.

Set it up in a few steps.

Add your Gemini API Key

Gemini gives the most natural, fluent translations. Your API Key stays on this device and is never uploaded.

No API Key yet? That's fine — you can skip it now and get one later; just pick free Google Translate on the next step.

Get a free API Key

Choose your default translator

Tap the floating button to translate

Open any web page and a floating translate button appears in the bottom-right corner. Tap it to translate the whole page, and tap again to switch back. You can drag it around, or resize or hide it in the settings.

Four-finger tap (alternative)

Four-finger tap is off by default. If you'd rather not see the floating button on screen, turn off the floating button and turn on the four-finger gesture in the settings — then tap any page with four fingers to translate, and tap again to switch back.

YouTube subtitles, auto-translated

When you open a YouTube video, Shinkansen translates the subtitles automatically. You can turn this off in the extension settings.

Last step: enable in Safari

How to enable

  1. Tap the button below to open Settings, then go to Safari → Extensions
  2. Turn on “Shinkansen” and set website access to “Allow” (“All Websites” recommended)

You can also tap the extensions button in Safari's address bar and choose “Manage Extensions”. The settings you just chose sync to the extension once it's enabled.
